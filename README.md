# ⚠️ Project Moved

**Click2Hide is now Click2Minimize.**

This open-source repository is deprecated and archived. We have migrated to a vastly improved commercial product with better performance, stability, and features.

Please visit our official website to download the latest version:

### [👉 click2minimize.com](https://click2minimize.com)

---

### Why the switch?
We developed Click2Minimize to solve the limitations of this original open-source project. The new app features:
- A completely new native macOS architecture.
- Advanced gestures support.
- Lock dock on main screen when multiple displays used.
- Make dock and trackpad a fidget to relief anxieties.
- Replace Finder with 3rd-party alternatives.
- Improved stability and resource usage.

Thank you to everyone who supported Click2Hide!
